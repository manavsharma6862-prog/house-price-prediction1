About This Project

Smart House Price Prediction System — ML + Django Capstone

Project Overview

This project implements a complete, end-to-end Machine Learning pipeline for predicting California house prices. It combines rigorous data science practices with a production-ready Django web application.

The system is trained on the California Housing Dataset (20,640 records, 8 features), applies feature engineering to create 11 total features, and compares three ML algorithms to automatically select the best model.

Key Pipeline Steps
1
Data Loading & Cleaning
Load raw California housing data, handle missing values
2
Missing Value & Outlier Handling
Remove anomalies using IQR method
3
Feature Engineering
Create 3 new derived features (rooms_per_person, bedrooms_ratio, income_per_room)
4
EDA & Visualization
Generate 5 professional matplotlib/seaborn plots
5
Model Training
Train Linear Regression, Random Forest & Gradient Boosting
6
Auto Model Selection
Best model chosen automatically by R² score
7
Django Deployment
Serve predictions via responsive web application
